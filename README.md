 # HOW TO USE NODE-RED (int this project)
 We are going to open NodeRed from the folder node-red
 1. Open Terminal in the folder
 ```sh
 cd node-red
 ```
 2. Install dependencies
 ```sh
 npm install
 ```
 3. Run node-red
 ```sh
 npm start
 ```
